# React Setup

Basic React project setup/boilerplate


# Pre-requisites

 - Node
 - VSCode

## Installation
Check existing node packages

    npm -g ls --depth=0

or locally (omit -g) :

    npm ls --depth=0

Install the following node mobdules globally

    npm i -g babel-eslint
    npm i -g eslint
    npm i -g eslint-plugin-react
    

## VSCode plugins

 - ESLint
 - Git History
 - GitLens
 - Cobalt2 Theme
 - Babel ES6/ES7
 - React Redux ES6 Snippets
 - React-beautify
Optional
 - Prettier
 - Code Runner
 - Live Server
